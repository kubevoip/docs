Install KubeVoIP with Helm, create SIP users and routes with the KubeVoIP CLI,
and register two SIP phones that can call each other.

Start with [Install KubeVoIP](/getting-started/install/), continue to the
[two-phone quickstart](/getting-started/quickstart/), and use
[Cleanup](/getting-started/cleanup/) when you are done with the test platform.
