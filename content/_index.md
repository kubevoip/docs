# KubeVoIP documentation

KubeVoIP is a Kubernetes-native SIP platform. It uses Kubernetes APIs to manage
SIP gateways, media relays, users, trunks, dial policies, routes, and Asterisk
application pods.

Start with the [two-phone quickstart](/getting-started/quickstart/), then read
the [networking guide](/networking/) before exposing SIP or RTP outside your
cluster.
